## ğŸš€ Getting Started

### Prerequisites
- Python 3.9+
- pip package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/TechWithAkash/DataVisualizer.git
cd DataVisualizer
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open your browser and navigate to:
```
http://localhost:5000
```
